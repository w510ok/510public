必须写在前面

本文件是基于公开招生考试信息、近年新高考数学命题导向、历年真题结构和高分卷常见作答方式整理的学习参考工具，不是江苏省教育考试院、教育部教育考试院、学校或教师发布的官方材料。

不保证提分、不提供押题、不提供所谓内部评分细则。数学成绩取决于试卷难度、个人基础、临场状态、阅卷尺度和规范作答等多因素。

考场红线

不得携带违规资料不得猜传试题不得使用违规电子设备不得冒险押题替代复习

临考最安全策略是：用“流程”稳定发挥，用“模板”组织步骤，用“检查”回收低级错误。

稳

先保基础盘

集合、复数、向量、三角、统计、直线圆等基础小题，目标是少丢不该丢的分。

厚

做厚步骤分

解答题不会做完，也要写定义域、设元、关系式、联立方程、单调表、结论句。

抢

压轴抢前半

最后两题不要空着；先做能落笔的前半问，别被整题难度吓住。

查

最后回收分

检查顺序：填涂/抄写 → 符号/范围 → 结论句 → 关键计算。

二、卷面结构坐标：先认清“分数池”

以下按近年江苏适用新高考全国卷、新课标卷公开结构和命题导向整理，最终以当年试卷、考试说明和监考要求为准。

选填保底池：概念 + 快速判断
中档主战池：常规模型 + 规范步骤
压轴回收池：前半问 + 可见推导
检查增益池：低级错少 1 个就是提分
题型板块临考定位中上生最优动作风险提醒
单选速度与稳定性先判断概念、定义域、数量级；能排除就不硬算。不要把小题当大题算，超过 2 分钟要标记跳过。
多选最容易“会但失分”逐项验证，先排错项，再处理可疑项；未证不选。不要凭感觉全选，也不要因为害怕只选一个。
填空结果准确性草稿纸保留短链路，答题卡只落最终值；写前复核范围。定义域、正负号、参数范围、端点最容易丢。
前几道解答题稳定拿分区题型模板化，结论句完整，尽量不跳步。这里丢分比压轴不会做更伤。
最后两道解答题分层拉开区不求全做完，优先写前半问和关键转化。不要把 20 分钟耗在一个没有进展的式子里。

三、考场 120 分钟流程：55分钟前求稳，55分钟后求厚

核心不是每题都做完，而是把可得分按顺序装入口袋。考场上最怕“前面纠结、后面空白、最后乱改”。

0—5′浏览全卷，标记必拿题、风险题、压轴前半问。
→
5—38′单选 + 容易多选 + 容易填空，先拿确定分。
→
38—55′回到多选和填空难点，最多二次处理，不恋战。
→
55—95′前几道解答题，按模板完整写步骤和结论。
→
95—112′最后两题抢前半问；写出设元、关系式、关键转化。
→
112—120′检查填涂、抄写、符号、范围、结论句。
正常版口令：55分钟前求“稳”，55分钟后求“厚”，100分钟后求“回收”。
应急版口令：若某题 3 分钟没有有效推进，立即写下已知转化并跳题，不能让一道题拖垮全卷。

四、六类解答题抢分模板

模板不是死背答案，而是帮助考场上快速组织“有效信息”。每道大题至少要让阅卷老师看见：你知道题目对象、方法入口和关键关系。

导数函数：定义域—求导—零点—单调—最值—结论

答题骨架： 1. 函数定义域为 D=____。 2. f'(x)=____。 3. 令 f'(x)=0，得临界点 x=____。 4. 讨论 f'(x) 在区间____上的符号，列单调性。 5. 因此 f(x) 在____处取得____，所以____。 抢分提醒：即使后面不会，定义域、导函数、临界点、单调区间也要写。

最值问题恒成立参数分类定义域不能漏

解析几何：设点设线—联立—韦达—目标量—化简

答题骨架： 1. 设点 A(x1,y1), B(x2,y2)，直线 l: y=kx+b 或 x=my+n。 2. 与圆锥曲线方程联立，得到一元二次方程____。 3. 由判别式 Δ>0，且 x1+x2=____，x1x2=____。 4. 将目标量（斜率、面积、距离、向量关系）转化为 x1+x2、x1x2、k、b。 5. 化简得到____，故结论为____。 抢分提醒：联立方程和韦达关系写出来，常常就是前半问关键步骤。

设而不乱中间变量分层别一行推到底

立体几何：关系证明—建系计算—角距体积

答题骨架： 证明型： 1. 因为____∥/⊥____，且____，所以____。 2. 根据线面/面面判定定理，得____。 建系型： 1. 以____为原点，____为 x 轴，____为 y 轴，建立空间直角坐标系。 2. 写出关键点坐标：A____, B____, C____。 3. 求向量/法向量，代入夹角或距离公式。 4. 故所求为____。 抢分提醒：坐标系建对、点坐标写对，后面计算错也可能保留部分步骤价值。

先关系后计算能建系就建系证明链别断

概率统计：定义事件—列分布—算期望—解释现实意义

答题骨架： 1. 设事件/随机变量 X 表示____。 2. 由题意可得 X 的可能取值为____。 3. 列分布列：P(X=____)=____。 4. E(X)=____，D(X)=____（如需要）。 5. 结合题意，结论为____。 抢分提醒：先把文字翻译成事件或随机变量，不要一上来套公式。

样本空间条件概率分布列表格优先

数列：识别模型—求通项—求和—证明/放缩

答题骨架： 1. 由递推式____，整理得____。 2. 判断为等差/等比/可裂项/可构造型。 3. 求得通项 an=____。 4. 若求和，采用公式/错位相减/裂项相消，Sn=____。 5. 若证明不等式，先写需要证明的目标，再放缩为____。 抢分提醒：数列题最怕“想到一点但不落笔”，把递推变形写出来就是入口。

等差等比错位相减裂项相消构造意识

三角/向量：公式转化—范围控制—几何意义

答题骨架： 三角： 1. 由题意和公式，化为 sin/cos/tan 的统一形式____。 2. 利用恒等变形或辅助角公式，得____。 3. 注意角的范围：____，所以____。 向量： 1. 设基底向量或坐标，表示目标向量____。 2. 用数量积/模长/夹角关系建立方程____。 3. 解得____。 抢分提醒：三角题最容易丢在角的范围；向量题最容易丢在数量积符号。

辅助角正余弦定理数量积范围必写

五、多选题专项：逐项验证，不凭感觉

步骤动作提醒
第一轮先找明显错误项定义域、极端值、特例、图像趋势常能排除。
第二轮对剩余项逐项写一句依据不能说明依据的选项，先标问号。
第三轮回看题干条件是否全部用到参数范围、正负、端点、是否“任意/存在”。
最后以试卷赋分说明为准原则：不把未证选项当确定项。

六、选填急救：小题快做但不能飘

集合：看清补集、交并、区间端点。

复数：先化 a+bi，再看模长/共轭。

向量：数量积符号、夹角范围最常错。

三角：先定象限，再套公式。

统计：均值、方差、百分位数不要凭感觉。

直线圆：距离公式、切线条件、圆心半径。

不等式：注意等号成立条件。

函数图像：定义域、奇偶、单调、端点。

小题口令：先看条件，再选方法；能代特殊值就别硬算；写答案前复核范围。

九、错题复盘：从“错题本”升级为“错因数据库”

普通错题本只抄答案，性价比很低。临考阶段只记录能改变下一次行为的信息。

审题错漏看“任意/存在”、范围、单位、端点
概念错定义域、单调性、独立事件、线面关系
方法错该建系不建系，该韦达不韦达
运算错符号、括号、移项、平方、根号
分类错参数讨论少一种情况
时间错小题恋战，压轴空白
错题复盘四句话： 1. 我错在：____。 2. 当时卡住的第一节点是：____。 3. 正确解法的第一个突破口是：____。 4. 下次遇到同类题，我先做：____。

十一、临考最简口令

选填口令

先条件，后方法；能排除不硬算；写答案前查范围。

解答口令

先写入口，再写转化；不会做完，也要让步骤分看得见。

压轴口令

不赌整题，抢前半；不恋战，留检查；能回收一分是一分。

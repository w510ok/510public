## 结论

1. 百度地图、Google地图可以通过官方 API 在线嵌入 HTML，但需要合法 AK/API Key、域名白名单、计费/配额和网络条件。
2. 不建议、也不在本包中实施：把百度地图、Google地图或 OSM 公共瓦片截图抓取、下载、缓存后打包进 PWA 离线发布。
3. 本包采用“原创内置地形沙盘图”作为离线底图：`assets/original-terrain.svg` 与 `assets/original-terrain.png`。该图用于教学推演，地形纹理和视觉表达为原创，点位和路线为史料研究基础上的近似复原。

## 可用方案

- 默认离线：内置示意地图 + 原创地形沙盘图。
- 国内在线：百度地图 JSAPI GL，输入浏览器端 AK，配置 Referer 白名单。
- 国际在线：Google Maps Static API，输入 API Key，启用 Maps Static API 和账单，页面按当前阶段生成在线静态图。

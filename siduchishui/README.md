# 四渡赤水战役关键决策沙盘推演（HTML + PWA）

## 文件说明

- `index.html`：主程序。内置示意地图、时间线、关键决策、推演器、依据与校核清单。
- `manifest.webmanifest`：PWA 配置。
- `service-worker.js`：离线缓存脚本。
- `assets/icon-192.png`、`assets/icon-512.png`：PWA 图标。

## 打开方式

直接双击 `index.html` 可以查看内置示意地图和推演器。若要启用 PWA 安装、离线缓存和百度地图脚本，建议在本文件夹中运行本地服务：

```bash
python -m http.server 8000
```

然后在浏览器打开：

```text
http://localhost:8000/index.html
```

## 百度地图接入

1. 到百度地图开放平台申请 Web 端 JavaScript API AK。
2. 打开本沙盘，点击“百度地图图层”。
3. 输入 AK 并点击“加载百度地图”。AK 仅保存在浏览器本地 `localStorage`，不会写入文件或上传。
4. 若地图无法显示，请检查 AK、服务配额、Referer 白名单与本地服务地址。

## 精度说明

本沙盘使用教学级近似坐标，适合课堂推演、管理复盘和研学准备；不适合作为测绘、导航或正式出版地图。正式使用前建议复核：

- 四渡赤水纪念馆馆藏战役要图；
- 地方志、文保点位与纪念碑现场坐标；
- 中央军委电文、亲历者日记和战役回忆录；
- 现代 GIS、百度地图/天地图/自然资源数据。

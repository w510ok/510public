# 金庸三书人生处世智慧｜互动研究 PWA

## 文件说明
- `index.html`：主页面，已内嵌样式与交互脚本，可直接打开浏览。
- `manifest.webmanifest`：PWA 配置。
- `service-worker.js`：离线缓存脚本。
- `assets/icon-192.png`、`assets/icon-512.png`：PWA 图标。

## 使用方式
1. 普通预览：直接双击 `index.html` 打开即可。
2. PWA/离线体验：在本文件夹中启动本地服务器，例如：
   ```bash
   python3 -m http.server 8080
   ```
   然后访问 `http://localhost:8080`。手机端可通过浏览器菜单“添加到主屏幕”。

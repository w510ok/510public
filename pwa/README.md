# 杯中有江湖：《笑傲江湖》酒趣研究 PWA

## 文件说明
- `index.html`：主页面，内置样式与交互脚本。
- `manifest.webmanifest`：PWA 应用清单。
- `sw.js`：离线缓存 Service Worker。
- `assets/icon-192.png`、`assets/icon-512.png`：应用图标。

## 使用方式
1. 直接双击 `index.html` 可作为普通 HTML 阅读。
2. 若要启用 PWA 安装与离线能力，请把整个文件夹部署到任意 HTTPS 静态站点，或本地用 `python3 -m http.server` 预览。
3. 首次访问后，浏览器会缓存页面；在支持 PWA 的浏览器中可添加到主屏幕。

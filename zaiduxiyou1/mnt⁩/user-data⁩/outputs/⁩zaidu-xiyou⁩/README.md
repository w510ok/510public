# 再读西游 · 配套资料包

## 包含文件

1. `zaidu_xiyou_data_base.xlsx`
   AI 研究数据底座：百回主题、八十一难、人物关系、妖怪分类、后台妖怪、法宝、专题证据、校核清单等。

2. `zaidu_xiyou_research_report.pdf`
   研究报告，适合归档、汇报与分享。

3. `zaidu_xiyou_reading_salon.pptx`
   读书会 / 讲座分享演示文稿。

4. `zaidu_xiyou_wechat_series.html`
   公众号连载母稿，含四篇专题文章。

5. `ai_classic_reading_method.md`
   AI 辅助名著深读方法模板，可复用到其它经典文本。

## 使用建议

- 对外展示：先发导览网页（`../index.html`）。
- 内部汇报：使用 PDF 报告与 PPT。
- 深入校核：打开 Excel 数据底座。
- 公众号连载：使用连载 HTML 中的四篇文章母稿。
- 方法复用：参考方法模板，迁移到《水浒传》《红楼梦》《水经注》《庄子》等。

## 声明

本项目以维基文库《西游记》百回本为可检索底本。正式公开发表前建议对照通行纸质校注本。AI 用于辅助建表、检索、归纳、图谱与初稿组织；观点与引用仍需人工校核。

# 水务行业宣贯沙盘｜HTML + PWA

## 使用方式

1. 解压后打开 `index.html` 即可离线浏览。
2. 部署到公司内网、公众号菜单、企业微信微站或静态服务器后，可作为 PWA 安装到手机桌面。
3. 首次联网打开后，浏览器会缓存页面、图标、SOP和参考清单；之后可离线使用。

## 内容定位

- 面向党委理论学习中心组、基层党支部、生产运行、水质检测、管网调度、工程建设、营销客服、合规董秘、信息化科创、社区物业用户代表等多角色。
- 参照《古田会议决议》的组织方法，转译为水务行业的“方向—组织—教育—宣传—纪律—群众工作—复盘纠偏”。
- 核心口号：用户至上，多供好水。

该工具为宣贯、培训和沙盘演练材料，不替代正式制度、应急预案、法律意见、证券信息披露文件或主管部门要求。涉及重大事项应走公司正式审批流程。

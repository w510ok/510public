专业调度系统 HTML 套件说明页

推荐理解方式：网格化调度是后台管控，解决统一视角和统一派单；岗位驾驶舱是移动端应用，解决岗位执行和现场回传；培训演练是围绕工作软件的场景化模拟，解决怎么练、怎么评、怎么复盘。

推荐使用顺序

  1. 先打开后台 Demo，讲清楚“为什么要一张图、事件总线、统一阈值和网格责任”。
  2. 再打开移动端 Demo，讲清楚“领导、总值班、泵站值守分别在手机上看什么、做什么”。
  3. 最后用培训演练页跑一遍场景，完成案例复盘、岗位 SOP 训练和班组评分。

二次开发建议

  • 正式项目应把 SCADA、GIS、视频平台、客服工单、资产台账接入统一事件中心。
  • 移动端应支持真正的定位签到、图片上传、语音备注、离线缓存和断点续传。
  • 后台应支持多图层筛选、事件回放、统计报表和演练脚本模板配置。
  • 演练模块可独立做成“课程 + 评分 + 复盘”子系统，支持班组积分和题库。

# AI辅助名著深读六步法

## 1. 建底本
确定一个可检索底本，并记录版本、作者、回目、引用边界。正式发表前保留纸质校注本复核机制。

## 2. 建数据表
至少建立六张表：百回主题表、人物关系表、八十一难表、妖怪/对象分类表、原文摘录表、专题观点表。

## 3. 做图谱
将数据表转为三类图：人物关系图、风险/后台矩阵、主题热力图。图谱服务理解，不为炫技。

## 4. 摘原文
每条摘录包含：回目、短摘、定位、主题、解释、用途、来源。以短摘为主，避免大段复制。

## 5. 写专题
从数据表和摘录表中抽取观点，先写小专题，再汇成研究报告。建议首批4-6篇。

## 6. 做发布包
形成 HTML/PWA、Excel数据底座、PDF报告、公众号稿、PPT讲稿、方法模板。发布前进行口径、引用、版权、事实四类校核。

## 推荐迁移对象
《水浒传》《三国演义》《红楼梦》《金瓶梅》《水经注》《庄子》《史记》、金庸小说等。

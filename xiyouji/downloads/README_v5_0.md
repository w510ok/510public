# 《再读西游》V5.0 正式发布包

## 包含文件

1. `zaidu_xiyou_v5_0_release.html`  
   HTML/PWA 主入口，可手机浏览、离线打开、打印转 PDF。

2. `zaidu_xiyou_data_base_v5_0.xlsx`  
   AI研究数据底座，包含百回主题、八十一难、人物关系、妖怪分类、后台妖怪、法宝、专题证据、校核清单等。

3. `zaidu_xiyou_research_report_v5_0.pdf`  
   正式研究报告，适合归档、汇报和分享。

4. `zaidu_xiyou_wechat_series_v5_0.html`  
   公众号连载母稿，含四篇首批专题文章。

5. `zaidu_xiyou_reading_salon_v5_0.pptx`  
   读书会/讲座分享PPT。

6. `ai_classic_reading_method_template_v5_0.md`  
   AI辅助名著深读方法模板，可复用到其他经典文本。

## 使用建议

- 对外展示：先发 HTML/PWA 主入口。
- 内部汇报：使用 PDF 报告与 PPT。
- 深入校核：打开 Excel 数据底座。
- 公众号连载：使用连载 HTML 中的四篇文章母稿。
- 方法复用：参考方法模板迁移到《水浒传》《红楼梦》《水经注》《庄子》等。

## 发布声明

本项目以维基文库《西游记》百回本为可检索底本。正式公开发表前建议对照通行纸质校注本。AI用于辅助建表、检索、归纳、图谱和初稿组织；观点与引用仍需人工校核。

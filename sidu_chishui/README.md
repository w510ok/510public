# 四渡赤水战役关键决策离线沙盘地图

版本：HTML + PWA 离线地图版  
制作：江南AI工作小组  
定位：内部学习研究、组织培训、战役沙盘推演、桌面投屏与手机阅读

## 本版调整

本版不再调用百度地图、Google 地图或其他在线地图 API，也不缓存第三方地图瓦片。地图工作台由以下内置图层构成：

1. 综合动态沙盘：原创 SVG 推演图，可随时间线切换阶段。
2. 四图对读总览：把三张附件地图与原创图合成一张对读看板。
3. 地形广域图：用户提供附件，用于观察川南、黔北、滇东与重庆方向的大空间关系。
4. 地形核心图：用户提供附件，用于观察赤水、习水、仁怀、遵义局部地形。
5. 战役示意图：用户提供附件，用于核对四渡赤水、南渡乌江、入滇、渡金沙江路线。
6. 原创底图：江南AI工作小组制作的教学推演底图。

## 使用方法

直接双击 `index.html` 可打开。若需要 PWA 安装和离线缓存，建议在文件夹内运行：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/index.html
```

部署到 GitHub Pages 后，访问：

```text
https://<账号>.github.io/<仓库名>/index.html
```

## 手机端阅读

手机端可横屏查看地图，也可竖屏阅读阶段卡片。图片图层支持放大、缩小和横向拖动；综合动态沙盘支持全域视野与核心区视野切换。

## 使用边界

本成果包用于内部学习研究和组织培训。第三张战役示意图图面可见“中国地图出版集团制”和审图号，公开出版、商用展陈、二次分发或外部宣传前，应另行核验授权与引用边界。

# 地图资料来源与使用边界说明

## 一、地图组成

本版地图工作台由四类图层组成：

- 用户提供的地形广域图；
- 用户提供的地形核心图；
- 用户提供的“中央红军四渡赤水河与南渡乌江示意图”；
- 江南AI工作小组制作的原创教学推演图和综合动态 SVG 沙盘。

## 二、为什么不再使用百度地图或 Google 地图

用户已明确要求不使用百度地图、Google 地图。因此本版删除外部地图 API 接入逻辑，不再加载在线脚本，不需要 AK/API Key，不依赖第三方底图服务。

## 三、公开发布边界

本成果包适合内部学习研究、组织培训、课堂沙盘和非商业研学讨论。若用于以下场景，应另行核验地图资料授权：

1. 正式出版物；
2. 收费课程或商业展陈；
3. 公开网站长期发布；
4. 微信公众号等外部宣传；
5. 作为地图产品、导览产品或测绘成果使用。

第三张战役示意图图面可见“中国地图出版集团制”和审图号 GS（2021）3573号，应保留原图标识，不得裁切掉版权、审图号和图例说明。

## 四、精度边界

本沙盘的原创 SVG 推演图为教学级示意图，不是测绘地图。正式研学路线、展陈施工、出版审校应使用权威党史军史资料、地方志、纪念馆战役图、文保点位和 GIS 坐标进一步复核。
